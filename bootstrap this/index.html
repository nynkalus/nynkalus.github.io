- About Bootstrap -

Bootsrap is a css framework that helps web develepors make responsive websites. Responsive meaning when you adjust the frame of the website the content of the site will keep a consistent look. To use bootstrap you can either link it from the internet or use a plug in that your coding software porvides. Bootstrap has does its styling on html but you can always override it in css

Browser Compatibility

Something that web developers cannot forget is that the web browser market is very fragmented and not all browsers interpret css correctly or the same. Bootstrap supports all of the modern web browsers as well as older ones

Active Development

Boostrap is under active development this means that new features, bug fixes and improvements are released frequently. This is important for choosing the software you use for a project since you do not want to use something that will eventually become out of date.

Flexibility

Bootstrap is an easy to use software for new web developers and only requires basic knowledge of html and css. New users will find Bootstrap accessible with its easily cusomizable colors, fonts, and paddings. Bootstrap even allows more skillful users to write new components through a css import to allow more efficient and customizable coding.

Community

The most helpful advantage for a new user is that there is an active community involved to help new users learn. There is a whole eco system deticated to Bootstrap with 3rd parties creating themes, templates, snippets, Java plugins, and more for you to get your hands on.
